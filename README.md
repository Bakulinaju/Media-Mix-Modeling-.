## Описание проекта
Цель проекта — понять, какие факторы реально влияют на продажи фармацевтического препарата, и можно ли на основе этих данных 
более осознанно планировать маркетинговые инвестиции.

Задача — на основе исторических данных по продажам, рекламной активности, ценам и сезонности построить Media Mix Model, которая:
- показывает вклад каждого фактора в продажи
- учитывает отложенный эффект рекламы и эффект насыщения
- позволяет построить понедельный прогноз продаж и использовать модель для принятия маркетинговых решений.

___

## Этапы работы над проектом
- **Подготовка данных:** загружены и подготовлены недельные данные по продажам, ТВ-рекламе бренда и конкурентов, а также ценам. Данные отсортированы по времени и очищены от пропусков и некорректных значений.
- **Формирование признаков:** созданы агрегированные признаки по группам ТВ-каналов, добавлены ценовые показатели, сезонные индикаторы и трендовые компоненты временного ряда.
- **Учет рекламынх эффектов:** к рекламной активности применены:
   - adstock — для учета отложенного эффекта рекламы,
   - saturation — для учета снижения отдачи при росте инвестиций.

- **Построение модели:** построена многофакторная линейная регрессионная модель (MMM) с интерпретируемыми коэффициентами.
  Модель обучена на историческом периоде и использована для прогноза продаж.
- **Оценка и визуализация:** рассчитаны метрики качества прогноза, построены графики динамики продаж, вкладов факторов, кривые насыщения и эластичности.

## Результат проекта
- Построена Media Mix Model, объясняющая динамику продаж с учетом рекламы, цен, сезонности и активности конкурентов.
- Получен понедельный прогноз продаж на 2013 год.
- Качество прогноза на прогнозном интервале:
  - SSR: 2.71 млрд
  - RMSE: 8 340 единиц продаж в неделю (в среднем модель ошибается примерно на 8.3 тыс. продаж в неделю)
- Модель показывает вклад каждого рекламного канала и наличие эффекта насыщения при росте инвестиций.

## Рекомендации
На основе результатов модели можно сделать следующие практические выводы:
1. Не наращивать ТВ-бюджет линейно — эффект рекламы со временем насыщается, и дополнительный бюджет дает все меньший прирост продаж.
2. Смещать фокус в каналы с более высокой отдачей на текущем уровне инвестиций, а не распределять бюджет равномерно.
3. Учитывать активность конкурентов, так как она оказывает заметное влияние на продажи.
4. Использовать модель для сценарного анализа — например, чтобы оценить, что произойдет с продажами при изменении рекламного бюджета или цены.
